\# Projeto Intercom DataOps - Jonathan Sienkiewicz



\## Objetivo

Demonstrar domínio do fluxo de dados entre a ferramenta de atendimento Intercom e um Data Lake, utilizando práticas reais de extração, transformação, armazenamento e visualização.



\## Problema

Empresas que utilizam o Intercom possuem grande volume de tickets e interações, porém não conseguem transformar esses dados em insights para decisão operacional.



\## Entregáveis (MVP)

\- Pipeline de extração de dados dos tickets

\- Limpeza e padronização das informações

\- Armazenamento estruturado

\- Dashboard analítico com métricas operacionais
