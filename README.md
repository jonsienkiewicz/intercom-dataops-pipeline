# Intercom DataOps Case Study

Projeto de portfólio focado em análise e estruturação de dados operacionais
extraídos da plataforma Intercom.

O objetivo é transformar dados brutos de tickets de atendimento em métricas
confiáveis para apoio à tomada de decisão operacional.

Os dados reais utilizados no projeto foram anonimizados e não fazem parte
deste repositório.
